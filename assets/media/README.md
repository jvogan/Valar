# Launch Media

This directory contains the small media bundle used by the public `Valar` GitHub repo:

- `hero.png` for the README hero image
- `cli-mcp-preview.png` for the agent and bridge workflow preview
- `app-preview.png` for the macOS app preview
- `social-preview.png` for the GitHub social preview image

Regenerate these files with:

```bash
python3 tools/generate_launch_media.py
```
