## What This Doc Set Covers

- a clean public landing page
- the two-command newcomer path: `make quickstart` and `make first-clip`
- the exact blessed model IDs and their support posture
- copy-paste install and synthesize commands
- the `make` and `tools/validate.sh` entrypoints for fresh-clone verification
- the required native prerequisites, including the Metal toolchain for CLI and daemon inference
- rough download and footprint expectations for the blessed families
- the public repo lineage and upstream dependencies
- a public-safe CI workflow that does not include internal archive jobs
- the public MCP bridge and advanced integration boundary
- the public maintainer and release checklist

## What This Doc Set Excludes

- private operator playbooks
- personal machine paths
- local SSD-specific setup
- experimental model families in the beginner path
- internal archive or corpus workflows
